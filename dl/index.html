Sensually Styled Definition Lists

Thoughts, have been simming in my head since the middle of the year 2003 when discussion on the CSS-Discuss about styling definition list (<dl>), on ways I could style a defintion list. I have come up with there examples; the first two are styled in much the manner of a table with the definition title being the left column and the defintition comprising the left column and the third is a more whimsical style.

First the example definition list that will used comes from a porition of the list of CSS Play examples.

Unformatted Example

Folder Tree List
Nested lists styled to look like a folder tree.
Drop Down Lllama Menu
A drop drop down menu that is triggered by hovering over a drop down marker instead of hovering over the menu label.
Title Block with a Shock
The use of the title tag as an extra blank tag set and the body tag as an extra container.
Reformat the Table
Use a table for its data structure but display in a non-table format.
Image Replacement—No Span
This example is based on the Background-Image to Replace Text method; but now with no extra span.
Combination of CSS Tricks for a Menu
A menu that uses for different CSS tricks to create a dynamic menu.
Multiple Columns of Equal Length
A three column layout with the columns of equal length.
Enlarger Button Menu

An example of a menu using icon that each icon will grow larger when hovering over it.

Mark Your Jump

Show users where they jumped in a page.

CSS Pop-Up Menus

CSS based menus that pop-up instead of down.
