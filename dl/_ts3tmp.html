Sensually Styled Definition Lists

Thoughts, have been shimmering in my head since the middle of the year 2003 when discussion on the CSS-Discuss about styling definition list (<dl>), on ways I could style a definition list. I have come up with three examples using a portion of the list of CSS Play examples. The first two are styled in much the manner of a table with the definition title being the left column and the definition comprising the left column and the third is a more whimsical style.

Example One

This first example is suppose to look like a two column table. Both the dd and the dt float to the left with an assigned width. The dt clears the left side to prevent it from wrapping around.

dl{ float: left; width: 97%; padding: 1%; margin-left: -1em; /*Push it over to the left.*/ margin-bottom: 1em; /*Matt Effect*/ border-width: 1em; border-style: solid; border-color: #EEF3F9; } dl dt{ float: left; clear: left; width: 30%; padding: .5em 5% 1em 5%; /*The -1px on the right margin pulls the dd over top of it by 1px.*/ /*The Overlap is because they both create the middle border and by overlapping them it does not matter if the dt or the dd are longer.*/ margin: 0 -1px 0 0; border-right: 1px solid #A5BFD6; font-weight: bold; } dl dd{ width: 50%; padding: .5em 4.5% 1em 5%; float: left; margin: 0 ; border-left: 1px #A5BFD6 solid; clear: right; /*Forces the dd to break to the next line.*/ } /*Creates the horizontal border between each *row*.*/ dl> dd + dt, dl > dd + dt + dd{ border-top: 1px #A5BFD6 solid; }
