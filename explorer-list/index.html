Third

Now I want to add folder next to the links that have children and page icons next to those that have no children. The links that are will have the page icon have no siblings. The links with the folder icon will have a sibling that is the nest list (<ul>). So I will style the links by default to have a folder icon and then to style those that have no sibling I will use :only-child.
