Image Replacement—No Span

Many of us have used the Background-Image to Replace Text method for inserting an image that is only really important to the layout through CSS. One of the downside of this method is the use of an extra span tag. But alas no more, by setting the overflow to hidden, the height to zero, the padding-top to the height of the image, and the background-image to the image you want to display you can get the same effect.

Example

Example Text for the Image Replacement

HTML

<h3 id="ex1">Example Text for the Image Replacement</h3>

CSS

#ex1{ /*Hide the text*/ height: 0; overflow: hidden; /*Size of the image*/ padding-top: 133px; width: 243px; /*The image*/ background-image: url(ex1.gif); background-repeat: no-repeat; }
